# NARA

NARA is a competitive token issuance protocol on Base.

Tokens are released only when ETH is committed.
Issuance halts automatically if participation drops to zero.

There is no presale and no free emissions.

This repository contains:
- Protocol documentation
- Mechanism and risk analysis
- Smart contracts (pre-deployment)

Status: pre-deployment  
Chain: Base (genesis)

Experimental software. No guarantees.
